# Deploying to Render

This backend is configured for deployment on [Render](https://render.com).

## Quick Start (Blueprint)

1. Push this repo to GitHub/GitLab/Bitbucket.
2. In [Render Dashboard](https://dashboard.render.com) → **Blueprints** → **New Blueprint Instance**.
3. Connect your repository and apply the `render.yaml` blueprint.
4. Add the required environment variables (see below).
5. Deploy.

## Environment Variables

Set these in **Render Dashboard → Your Web Service → Environment**:

| Variable | Required | Description |
|----------|----------|-------------|
| `DATABASE_URL` | Yes (auto) | Provided by Render when linked to a PostgreSQL database |
| `SECRET_KEY` | Yes (auto) | Generated by Render if using blueprint |
| `CLOUDINARY_URL` | Yes | From [Cloudinary Dashboard](https://cloudinary.com/console) (e.g. `cloudinary://key:secret@cloud_name`) |
| `CORS_ALLOWED` | Yes | Comma-separated frontend origins (e.g. `https://dicentra-frontend.vercel.app,https://yoursite.com`) |
| `EMAIL_HOST` | Yes | SMTP server for password reset emails |
| `EMAIL_HOST_USER` | Yes | SMTP username |
| `EMAIL_HOST_PASSWORD` | Yes | SMTP password |
| `EMAIL_DOMAIN` | Yes | Frontend domain for reset links (e.g. `https://yoursite.com`) |
| `DEBUG` | No | Set to `false` in production (default) |

## Manual Setup

If not using the blueprint:

1. Create a **PostgreSQL** database on Render.
2. Create a **Web Service** → connect repo → select **Python 3**.
3. Set **Build Command**: `./build.sh`
4. Set **Start Command**: `gunicorn config.wsgi:application --bind 0.0.0.0:$PORT`
5. Add all environment variables above.
6. Link the database so `DATABASE_URL` is available.

## After Deploy

- **Create admin user**: Render Dashboard → Shell → run `python manage.py createsuperuser`
- **Update frontend**: Set your frontend's API base URL to `https://<your-service>.onrender.com`
- **CORS**: Ensure `CORS_ALLOWED` includes your frontend's exact origin URL.

## Build vs Start

- **Build** (`build.sh`): Installs deps, runs `collectstatic`, runs migrations.
- **Start**: Launches Gunicorn. Render sets `$PORT` automatically.
